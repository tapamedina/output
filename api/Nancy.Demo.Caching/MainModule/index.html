After AfterPipeline

The post-request hook

The post-request hook is called after the response is created by the route execution. It can be used to rewrite the response or add/remove items from the context.

This is automatically set by Nancy at runtime.
Inherited from NancyModule
Before BeforePipeline

The pre-request hook

The PreRequest hook is called prior to executing a route. If any item in the pre-request pipeline returns a response then the route is not executed and the response is returned.

This is automatically set by Nancy at runtime.

The error hook

The error hook is called if an exception is thrown at any time during executing the PreRequest hook, a route and the PostRequest hook. It can be used to set the response and/or finish any ongoing tasks (close database session, etc).

This is automatically set by Nancy at runtime.
